# HealthCare-MIS

A Full Stack MERN Healthcare Management Information System (MIS) developed to manage healthcare staff, departments, employees, leave requests, and user management efficiently.

---

## Features

* Admin Authentication & Authorization
* Dashboard Management
* Add / Update / Delete Departments
* Add / Update / Delete Employees
* Manage Employee Leave Requests
* User Management System
* Change User Roles & Status
* Attendance & Salary Management
* Protected Routes with JWT
* Responsive UI Design
* REST API Integration
* Real-Time Notifications

---

## 🛠️ Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* CSS / Bootstrap / Tailwind CSS
* React Toastify

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/HealthCare-MIS.git
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
